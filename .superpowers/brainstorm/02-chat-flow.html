這是對話式引導的示意圖。請確認流程是否正確：

① 接機/送機 → ② 日期+時間 → ③ 航班(選填) → ④ 車型 → ⑤ 乘客數+行李數 → ⑥ 聯絡人+電話 → ⑦ 上/下地點（依接/送機自動設定機場端）→ ⑧ 費用模式 → ⑨ 金額 → ⑩ 摘要確認 → ⑪ 發單

⑨金額：「客下轉帳」模式 → 問「司機實拿金額」
⑨金額：「代收」模式 → 問「司機代收現金金額」+「回金給派單人金額」
