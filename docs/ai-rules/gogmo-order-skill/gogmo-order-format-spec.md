# gogmo 標準派單格式 v1.0

> 這份文件是給**派單方**看的派單規範。  
> AI 解析引擎依照此規範運作 —— 符合規範的訊息會被快速派出，不符合的會被退回修正。

---

## 為什麼有這份規範

在 LINE 群裡，你需要重貼訊息怕被洗掉、寫「一套不拆」怕司機只接一趟、用各種縮寫趕著貼單。

**在 gogmo 上，這些行為都不需要了**：

| LINE 群裡的痛 | gogmo 平台的解法 |
|---|---|
| 訊息被洗掉 → 重貼 | 訂單卡常駐，未接單會自動提醒 |
| 一套不拆 → 寫在訊息裡求司機 | 平台「綁定派單」按鈕，司機接一就接全套 |
| 要讓司機看到 → 用 emoji 與 ★ 裝飾 | 訂單卡按時間自動排序，重要訊息有專屬欄位 |
| 規則散落各處 → 寫在訊息角落 | 「特殊需求」「結帳方式」獨立欄位 |

所以你**不需要照舊習慣寫訊息**。照下面格式貼，系統會更快幫你派出去。

---

## 最小可接受格式

### 單筆訂單（最簡）

```
日期 時間 類型 地點 價格
```

✅ 範例：
```
4/19 22:10 接機 大安區信義路 850
4/20 07:00 送機 台中南屯 1500
```

### 完整版（推薦）

```
日期 時間 類型 地點 [車型] [航班號] 價格
```

✅ 範例：
```
4/19 22:10 接機 大安區信義路 小車 CI920 850
4/20 07:00 送機 台中南屯 大車 1500
```

---

## 五大必填欄位

| 欄位 | 規則 | ✅ 接受 | ❌ 不接受 |
|---|---|---|---|
| **日期** | 月/日 或 完整日期 | `4/19`、`2026/4/19` | `明天`、`後天`、`下週一` |
| **時間** | 24 小時制 HH:MM | `22:10`、`07:00`、`0:30` | `早上`、`晚上`、`七點半` |
| **類型** | 接機 / 送機 / 包車 / 接駁 | `接機`、`送機`、`接`、`送` | `R`、`配車`、空白 |
| **地點** | 區或更具體 | `大安區`、`大安區信義路` | `市區`、`隨便`、空白 |
| **價格** | 純數字（單位：新台幣） | `850`、`1500` | `便宜一點`、`看情況` |

> **接機/送機的隱含規則**：  
> 「接機」預設機場 = 桃園機場（TPE），終點 = 你寫的地點  
> 「送機」預設機場 = 桃園機場（TPE），起點 = 你寫的地點  
> 如果是松山（TSA）、高雄小港（KHH）、台中（RMQ），請明確標註：`4/19 22:10 接機(TSA) 大安區 850`

---

## 三大選填欄位

| 欄位 | 規則 | 範例 |
|---|---|---|
| **車型** | 小車 / 大車 / 休旅 / 九座 | `小車`、`大車` |
| **航班號** | 航空代碼 + 航班號 | `CI920`、`BR189` |
| **特殊需求** | 用括號標註 | `(無菸)`、`(高爾夫球具*4)`、`(指定 G 獨立)` |

---

## 多筆訂單（同一則訊息）

每筆訂單**獨立一行**，用空行分隔：

✅ 正確：
```
4/19 22:10 接機 大安區信義路 850

4/19 22:30 接機 新店安民街 900

4/19 22:35 接機 內湖行善路 900
```

❌ 錯誤（多筆塞一行）：
```
4/19 22:10接大安22:30接新店22:35接內湖
```

## 套裝訂單（綁定派單）

如果你希望司機**一次接下整套**（例：早上送機 + 下午接機 = 同一台車），**不要**寫「一套不拆」。

請在 gogmo App 上：
1. 先建立第一筆訂單
2. 點「綁定下一筆」
3. 系統會把整套標為 `🔗 綁定派單`，司機接一即接全套

**為什麼這樣比較好**：
- 司機看到「綁定派單」標記，會評估能否完整接下
- 系統自動防止只接一半的狀況
- 你不用在訊息裡寫額外說明

---

## ❌ 不接受的格式（會被退回）

| 違規類型 | 例子 | 為什麼不行 |
|---|---|---|
| 模糊時間 | `早上、晚上、七點多` | AI 無法判斷具體時間 |
| 相對日期 | `明天、後天、下週` | 會在跨日訊息中出錯 |
| 隱含地點 | `0:05三重送`（沒寫日期） | 缺日期無法排程 |
| 範圍價格 | `1500-2000`、`看情況` | 司機無法決定是否接 |
| 一行多筆 | `5:00信義/北投/南港 各900` | 解析容易出錯 |
| 「一套不拆」 | 寫在訊息文字裡 | 請用平台「綁定派單」功能 |
| 重複貼單 | 同一筆訂單貼 5 次 | 平台有自動催單，重貼會被收費 |
| 純廣告 | `✈️包車找我❤️` | 不是訂單 |

---

## 違規處理機制

| 次數 | 處理 |
|---|---|
| 第 1 次 | AI 自動退回，附上「**改寫成標準格式的版本**」給你參考 |
| 第 2 次（同類型違規） | 系統警告 + 規範連結 |
| 第 3 次（同類型違規） | 暫停派單權限 24 小時 |
| 重複貼單 | 第 2 次起每筆收取 NT$5 處理費 |

> **AI 改寫範例**  
> 你貼：`0:05三重送`  
> AI 回：「⚠️ 這則訊息缺日期。下次請寫成：`4/19 00:05 送機 三重區 700` ← 點此複製」

## FAQ

**Q: 我習慣用 LINE 貼，能不能照舊？**  
A: 短期內可以，AI 會盡量解析。但 AI 解析完會把訂單**改寫成標準格式**回給你看 —— 久了你就會習慣了。重要的是，照標準格式貼，**訂單能更快派出去**（不用 AI 反覆確認）。

**Q: 司機指定要求（無菸、會說台語等）怎麼寫？**  
A: 用括號標註在地點後面：`4/19 22:10 接機 大安區 (無菸司機) 850`。或者更建議：在 gogmo App 上設定「司機偏好」，不用每次都打字。

**Q: 我有 100 筆訂單一起貼會不會被當廣告？**  
A: 不會，但建議用 App 的「批次匯入」功能，可以從 Excel 直接貼。

**Q: 預設價格能不能設定一次套用所有？**  
A: 可以。在 App 設定「派單偏好」，預設「該批次預設價 = $700」，個別訂單只標特殊價即可。
