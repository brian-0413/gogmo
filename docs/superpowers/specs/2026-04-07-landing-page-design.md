# Landing Page 設計規格

**日期**：2026-04-07
**功能**：首頁（登入前 landing page）
**檔案**：`docs/superpowers/specs/2026-04-07-landing-page-design.md`

---

## 1. 設計方向

**目標對象**：機場接送司機、派單方（車頭）
**風格**：極簡清新 Airbnb 風，米黃暖色背景
**核心概念**：左右雙卡片佈局 — 左側輪播區（司機/派單方切換）+ 右側即時數據牆

---

## 2. 版面結構

### 2.1 Navigation（固定頂部）
- 左：goGMO 品牌 logo
- 中：司機服務 / 派單方服務（錨點連結）
- 右：登入按鈕（#FF385C 背景，白色文字，圓角 8px）

### 2.2 Hero Section（雙卡片佈局）
- **整體背景**：#FAF8F5（暖米黃色）
- **包裝**：max-width 1200px，水平對齊，間距 16px

#### 左卡 — A Card（輪播區）
- 背景：白色
- 圓角：24px
- 最小高度：520px
- 內距：40px
- 內容：eyebrow + 標題 + 說明 + CTA 按鈕 + 標籤列
- 內含漸層裝飾（右上角 radial-gradient）
- **輪播行為**：
  - 2 張投影片（司機視角 / 派單方視角）
  - 由左而右滑動（translateX），進場從右側滑入，離場往左側滑出
  - 自動輪播 4 秒
  - 底部：dot indicators + 左右箭頭按鈕

#### 右卡 — B Card（即時數據牆）
- 背景：#FFF5F0（淡奶橘）
- 圓角：24px
- 最小高度：520px
- 內距：32px
- 內容：
  - Header：「即時動態」+ 呼吸燈圓點（#FF385C）
  - 4 格數據卡片（2×2 grid）
    - 司機接單動態（urgent 樣式，紅色數值）
    - 所有訂單
    - 已派出
    - 註冊司機
  - 最新派出 feed（3 筆即時訂單流）

### 2.3 Feature Sections（功能特色）
分兩區：

#### 司機專區（白色背景）
- 標題：司機的接單樂園
- 4 項功能（01–04 編號，大字透明度 0.3）：
  1. AI 智能排班推薦
  2. 3 秒即時推播
  3. 手機一鍵完成
  4. 帳務管理中心
- 每項含說明 + feature tags（藍底白字）
- 右上 CTA 按鈕

#### 派單方專區（#FAF8F5 背景）
- 標題：車頭的派單天堂
- 4 項功能（01–04 編號，透明度 0.2）：
  1. AI 智慧解析，多筆行程一次生成
  2. 行控中心即時掌握
  3. 帳務結算效率升級
  4. 降低成本
- 每項含說明 + feature tags（橙底白字）
- 右上 CTA 按鈕

### 2.4 CTA Banner
- 背景：深色（#1C1917）
- 標題 + 說明 + 雙 CTA 按鈕（主色 / 次色）
- 水平置中

## 5. 實作技術

- 框架：Next.js 14 App Router（`src/app/page.tsx`）
- 樣式：Tailwind CSS
- 輪播：純 CSS transitions + vanilla JS（translateX 動畫）
- 即時數據：SSE polling（3 秒）到現有 API `/api/drivers/events`
- RWD：mobile-first，`sm:` / `md:` / `lg:` breakpoints
