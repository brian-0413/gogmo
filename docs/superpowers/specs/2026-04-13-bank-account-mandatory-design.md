# 銀行帳號列為註冊必填 — 規格文件

## 1. 概述

將銀行帳號（銀行代碼 + 銀行帳號）列為司機註冊的必填項目，並做為接單前的必要條件。派單方（Dispatcher）不需要填寫銀行帳號。

### 目標
- 司機註冊時必須填寫銀行代碼與帳號
- 未填寫銀行帳號的司機無法接單（帳號審核通過後仍受此限制）
- 銀行代碼採用下拉選單，避免輸入錯誤

## 4. API 變動

### 註冊 API — `POST /api/auth`
- 司機註冊時接受 bankCode / bankAccount 參數
- 寫入 `Driver.bankCode` 和 `Driver.bankAccount`
- 若未填寫則 reject（司機必填）

### 接單 API — `POST /api/orders/[id]/accept`
- 在 `accountStatus === 'ACTIVE'` 檢查之後，加入：
  - `Driver.bankCode IS NOT NULL`
  - `Driver.bankAccount IS NOT NULL`
- 缺少任一 → 回傳 400：「請先至個人中心填寫銀行帳號」
